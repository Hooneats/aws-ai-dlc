# 페르소나 정의

## 고객 페르소나

### 페르소나 1: 김민준 (디지털 네이티브 고객)
- **나이**: 28세 | **직업**: IT 회사 직원 | **디지털 숙련도**: 높음
- **특징**: 빠른 주문 선호, 메뉴를 빠르게 탐색하고 즉시 결정
- **동기**: 대기 시간 없이 빠르게 주문
- **불만 요소**: 느린 UI, 복잡한 주문 과정
- **시나리오**: 점심시간에 동료와 식당 방문, 빠르게 주문

### 페르소나 2: 박영숙 (디지털 미숙 고객)
- **나이**: 62세 | **직업**: 은퇴자 | **디지털 숙련도**: 낮음
- **특징**: 큰 글씨와 명확한 버튼 선호, 메뉴를 천천히 살펴봄
- **동기**: 직원 도움 없이 스스로 주문
- **불만 요소**: 작은 글씨, 복잡한 화면, 실수로 잘못 누르는 것
- **시나리오**: 가족 모임으로 식당 방문, 천천히 메뉴 탐색

### 페르소나 3: 최수진 (단체 주문 고객)
- **나이**: 35세 | **직업**: 회사원 (팀장) | **디지털 숙련도**: 중간
- **특징**: 여러 명의 주문을 취합하여 주문, 추가 주문 빈번
- **동기**: 여러 사람의 주문을 실수 없이 정확하게 처리
- **불만 요소**: 주문 내역 확인 어려움, 추가 주문 시 혼동, 총 금액 파악 불편
- **시나리오**: 팀 회식 8명, 여러 차례 추가 주문하며 전체 내역/총액 확인

## 관리자 페르소나

### 페르소나 4: 이정호 (매장 사장님)
- **나이**: 45세 | **직업**: 식당 운영자 (자영업) | **디지털 숙련도**: 중간
- **특징**: 매장 전체 운영 관리, 실시간 주문 현황 파악 필요
- **동기**: 인건비 절감, 주문 실수 감소, 효율적 매장 운영
- **불만 요소**: 주문 누락, 테이블 상태 파악 어려움, 수기 관리 비효율
- **시나리오**: 피크 시간에 대시보드 모니터링하며 주문 상태 관리
