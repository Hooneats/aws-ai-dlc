# 🧑‍💻 고객 스토리

## Epic 1: 테이블 자동 로그인

### US-C01: 테이블 태블릿 자동 로그인
**As a** 고객, **I want** 별도 로그인 없이 바로 주문 화면을 보기를, **So that** 즉시 주문할 수 있다.

**GWT**: Given 초기 설정 완료 상태 → When 브라우저 접속 → Then 자동 로그인 후 메뉴 화면 표시

**체크리스트**: 로그인 정보 로컬 저장 | 브라우저 재시작 시 유지 | 실패 시 초기 설정 화면 | 세션 ID 생성
**엣지 케이스**: 로컬 저장소 초기화, 네트워크 불안정

---

## Epic 2: 메뉴 조회 및 탐색

### US-C02: 추천 메뉴 확인
**As a** 고객, **I want** 메뉴 상단에서 추천 메뉴를 보기를, **So that** 빠르게 선택할 수 있다.

**GWT**: Given 추천 메뉴 지정 상태 → When 메뉴 화면 접속 → Then 상단에 추천 메뉴 섹션 표시

**체크리스트**: 카테고리 위에 표시 | 추천 없으면 섹션 미표시 | 장바구니 추가 가능

### US-C03: 카테고리별 메뉴 조회
**As a** 고객, **I want** 카테고리별로 메뉴를 탐색하기를, **So that** 원하는 메뉴를 빠르게 찾을 수 있다.

**GWT**: Given 로그인 완료 → When 메뉴 화면 접속 → Then 카테고리별 메뉴 카드 표시

**체크리스트**: 기본 화면으로 표시 | 카테고리 탭 이동 | 메뉴명/가격/설명/이미지 표시 | 부드러운 전환
**엣지 케이스**: 빈 카테고리, 이미지 로딩 실패

### US-C04: 메뉴 상세 및 할인/품절 확인
**As a** 고객, **I want** 메뉴 상세와 할인/품절 상태를 보기를, **So that** 정확히 파악하고 주문할 수 있다.

**GWT**: Given 메뉴 목록 표시 → When 메뉴 카드 터치 → Then 상세 정보+할인/품절 표시

**체크리스트**: 메뉴명/가격/설명/이미지 표시 | 할인: 할인율+원래가격(취소선)+할인가 | 품절: 주문 불가 | 장바구니 추가 가능

## Epic 3: 장바구니 관리

### US-C05: 장바구니에 메뉴 추가
**As a** 고객, **I want** 메뉴를 장바구니에 담기를, **So that** 여러 메뉴를 한 번에 주문할 수 있다.

**GWT**: Given 메뉴 화면 → When 추가 버튼 터치 → Then 장바구니 추가+총 금액 업데이트

**체크리스트**: 수량 표시 | 동일 메뉴 수량 증가 | 할인가 계산 | 품절 추가 불가 | 실시간 금액 계산 | 로컬 저장(새로고침 유지)

### US-C06: 장바구니 수량 조절 및 삭제
**As a** 고객, **I want** 수량 조절/삭제하기를, **So that** 정확한 수량으로 주문할 수 있다.

**GWT**: Given 장바구니에 메뉴 존재 → When 수량 변경/삭제 → Then 수량 변경+총 금액 재계산

**체크리스트**: 증가/감소 버튼 | 0이면 삭제 | 개별 삭제 | 전체 비우기 | 즉시 재계산

## Epic 4: 주문 생성

### US-C07: 주문 확정
**As a** 고객, **I want** 요청사항 입력 후 주문 확정하기를, **So that** 빠르게 주문 완료할 수 있다.

**GWT**: Given 장바구니에 메뉴 존재 → When 주문 확정 터치 → Then 주문 전송+주문번호 표시+메뉴 화면 이동

**체크리스트**: 최종 확인 화면 | 요청사항 입력(선택) | 주문번호 표시 | 장바구니 자동 비우기 | 메뉴 화면 리다이렉트 | 실패 시 에러+장바구니 유지
**엣지 케이스**: 네트워크 오류, 주문 중 품절(서버 검증)

### US-C08: 추가 주문
**As a** 고객, **I want** 기존 주문 후 추가 주문하기를, **So that** 필요할 때마다 추가할 수 있다.

**GWT**: Given 이전 주문 완료 → When 새 메뉴 선택+주문 확정 → Then 새 주문 생성+같은 세션 연결

**체크리스트**: 독립적 새 주문 생성 | 같은 세션 ID 연결 | 주문 내역에서 전체 확인

## Epic 5: 주문 내역 조회

### US-C09: 현재 세션 주문 내역 조회
**As a** 고객, **I want** 현재 세션 주문 내역을 보기를, **So that** 전체 현황과 총 금액을 파악할 수 있다.

**GWT**: Given 주문 존재 → When 주문 내역 화면 접속 → Then 현재 세션 주문 시간순 표시

**체크리스트**: 주문번호/시각/메뉴/수량/금액/상태 표시 | 시간순 정렬 | 현재 세션만 | 정산 완료 제외

### US-C10: 주문 상태 실시간 확인
**As a** 고객, **I want** 주문 상태가 실시간 업데이트되기를, **So that** 음식이 언제 나오는지 알 수 있다.

**GWT**: Given 주문 내역 표시 → When 관리자가 상태 변경 → Then SSE로 실시간 반영

**체크리스트**: 고객에게는 "주문완료"로 통합 표시 (대기중/준비중/완료 구분 없음) | SSE 연결 유지 | 취소된 주문은 목록에서 제외

## Epic 6: 서비스 호출

### US-C11: 서비스 요청
**As a** 고객, **I want** "서비스 요청" 카테고리에서 직원 호출/물/수저 등을 선택하기를, **So that** 직원을 찾지 않고 서비스를 받을 수 있다.

**GWT**: Given 메뉴 화면에서 "서비스 요청" 카테고리 선택 → When 서비스 항목 터치 → Then 관리자에게 실시간 전달

**체크리스트**: "서비스 요청" 카테고리가 메뉴 카테고리 탭에 표시 | 항목 목록(직원/물/수저/앞접시/냅킨 등) | 완료 피드백 | 중복 요청 가능 | 장바구니에 담기지 않고 즉시 요청

**체크리스트**: 상태 변경(대기중→준비중→완료) | 고객 화면 SSE 실시간 반영 | 상태별 색상 구분

**GWT**: Given 테이블에 주문 존재 → When 정산 버튼+확인 → Then 주문 이력 이동+현재 주문/총액 리셋

**체크리스트**: 정산 확인 팝업 | 주문 내역→과거 이력 이동 | 현재 주문/총액 0 리셋 | SSE로 태블릿 장바구니 초기화+화면 리셋 | 새 세션 시작 가능 | 성공/실패 피드백

### US-A08: 과거 주문 내역 조회

**GWT**: Given 테이블 관리 화면 → When 과거 내역 버튼 클릭 → Then 과거 주문 목록 시간 역순 표시

**체크리스트**: 테이블별 과거 주문 목록 | 주문번호/시각/메뉴/총금액/정산시각 | 날짜 필터링 | 닫기→대시보드 복귀

### US-A10: 추천 메뉴 지정

**GWT**: Given 메뉴 목록 → When 품절 설정/해제 → Then 고객 화면에서 주문 불가 처리

## Epic 12: 매출 통계 및 운영 대시보드

### US-A16: 실시간 운영 현황 대시보드
**As a** 관리자, **I want** 오늘의 실시간 운영 현황을 한눈에 보기를, **So that** 매장 상태를 즉시 파악할 수 있다.

**GWT**: Given 통계 화면 접속 → When 대시보드 탭 선택 → Then 오늘 실시간 현황 표시

**체크리스트**: 오늘 누적 매출/주문 건수 | 영업 중 테이블 수/전체 테이블 수 | 평균 객단가 | 인기 메뉴 TOP 5 | 시간대별 매출 추이 그래프

### US-A17: 일별 매출 통계 조회
**As a** 관리자, **I want** 특정 날짜의 매출 통계를 보기를, **So that** 일별 매출을 분석할 수 있다.

**GWT**: Given 통계 화면 → When 날짜 선택 → Then 해당 날짜의 매출 통계 표시

**체크리스트**: 총 매출+총 주문 건수 | 테이블별 매출/주문 건수 | 메뉴별 판매 수량/매출액(판매량 순)

### US-A18: 기간별 매출 통계 조회
**As a** 관리자, **I want** 기간별 매출 추이를 보기를, **So that** 장기적인 매출 패턴을 파악할 수 있다.

**GWT**: Given 통계 화면 → When 시작일~종료일 선택(최대 1년) → Then 기간별 매출 통계 표시

**체크리스트**: 기간 내 총 매출/총 주문/평균 일 매출 | 일별 매출 추이 그래프 | 기간 내 메뉴별 판매 순위 | 기간 내 테이블별 매출 순위 | 요일별 매출 패턴
